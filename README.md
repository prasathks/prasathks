### 🚀 Featured Projects

#### [NexQ — Real-Time Digital Queue Intelligence System](https://github.com/prasathks)
*A full-stack platform for hospitals, banks, and service centers to eliminate physical waiting lines.*
- **Tech Stack:** Java 17, Spring Boot, JWT, MySQL, Hibernate, WebSocket
- **Highlights:** Built FIFO-based queue processing with thread-safe token allocation and real-time "Now Serving" updates via WebSocket/SSE. Implemented JWT role-based access control.

#### [Grid Guard — AI-Powered Smart Grid Fault Detection System](https://github.com/prasathks/grid-guard-fault-detection)
*An edge AI system to detect live conductor snaps and High-Impedance Faults (HIF) in low-voltage distribution.*
- **Tech Stack:** ESP32, TensorFlow Lite Micro, Python, LoRa
- **Highlights:** Deployed a quantized CNN model on ESP32 achieving 98.7% accuracy with 77ms response time. Implemented a Self-Healing LoRa Mesh Network covering up to 4.8km.

#### [Forgeflow ERP — Industrial Worker Attendance & Payroll Management](https://github.com/prasathks)
*A responsive web-based payroll management system for manufacturing units.*
- **Tech Stack:** HTML5, CSS3, JavaScript, Google Apps Script
- **Highlights:** Integrated Google Sheets as a real-time backend via Google Apps Script, enabling live data management for 100+ workers without a traditional database server.
